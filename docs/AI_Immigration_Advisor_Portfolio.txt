AI Immigration Advisor (ML + DL + LLM Portfolio)

프로젝트 개요

AI 기술을 활용하여 사용자에게 가장 적합한 이민 국가를 추천하고, 비자
승인 가능성을 예측하며, 이민 관련 상담을 제공하는 AI 기반 이민 플랫폼

------------------------------------------------------------------------

핵심 기술 스택

Machine Learning

-   비자 승인 가능성 예측 (Classification)
-   국가 추천 시스템
-   생활비 예측 (Regression)
-   취업 가능 직군 추천

추천 알고리즘 - Random Forest - XGBoost - LightGBM - CatBoost

------------------------------------------------------------------------

Deep Learning

-   OCR 기반 여권/비자/증명서 자동 인식
-   문서 종류 분류
-   자동 정보 추출

------------------------------------------------------------------------

LLM

-   AI 이민 상담 챗봇
-   RAG 기반 국가별 이민 정책 검색
-   사용자 조건 분석
-   부족한 조건 및 개선 방향 제시
-   체크리스트 자동 생성

------------------------------------------------------------------------

주요 기능

1.  AI 국가 추천
2.  비자 승인 확률 예측
3.  서류 OCR 자동 입력
4.  공식 자료 기반 AI 상담(RAG)
5.  국가별 생활비 예측
6.  취업 가능 직군 추천
7.  부족한 조건 분석
8.  이민 준비 체크리스트 생성

------------------------------------------------------------------------

예시 서비스 흐름

사용자 입력 - 나이 - 학력 - 경력 - 영어 점수 - 직업 - 희망 국가
